# 꾸미기 상점 탭

## 개요

기존 상점(`#shop-overlay`)에 꾸미기 탭을 추가한다.  
이모트 / 프로필 배경 / 뱃지를 코인으로 구매하고, 보유 아이템을 장착한다.

---

## 탭 구조

기존 상점 탭에 꾸미기 탭 추가:

```
[스킨] [꾸미기]
```

꾸미기 탭 내부 서브탭:
```
[이모트] [배경] [뱃지]
```

---

## 아이템 카드 UI (서브탭 공통)

각 아이템을 카드로 표시:

```
┌──────────────┐
│   이모지/미리보기  │
│    아이템 이름    │
│    💰 300코인   │
│  [구매] or [장착] │
│   또는 [장착됨]   │
└──────────────┘
```

| 상태 | 버튼 | 동작 |
|---|---|---|
| 미보유 | 구매 (N코인) | 코인 차감 → ownedXxx 추가 → 즉시 장착 |
| 보유 + 미장착 | 장착 | equippedXxx 변경 |
| 보유 + 장착 중 | 장착됨 (비활성) | 없음 |
| 코인 부족 | 구매 (N코인, 흐리게) | 클릭 시 "코인이 부족합니다" 토스트 |

---

## 구매 플로우

1. `구매` 버튼 클릭
2. 코인 잔액 확인
   - 부족: 토스트 메시지 "코인이 부족합니다" 표시 (2초)
   - 충분: 코인 차감 → 아이템 `ownedXxx` 배열에 id 추가 → `equippedXxx`에 장착 → 버튼 "장착됨"으로 전환
3. `account` localStorage 저장

---

## 장착 플로우

1. `장착` 버튼 클릭
2. `equippedXxx` = 해당 id
3. 동일 카테고리 기존 장착 아이템 → "장착" 버튼으로 전환
4. 프로필 카드 / 이모트 즉시 반영
5. `account` localStorage 저장

---

## 코인 표시

- 상점 상단에 현재 코인 잔액 표시 (기존 스킨 탭과 동일)
- 구매 후 잔액 즉시 갱신

---

## Acceptance Criteria

- [ ] 상점에 꾸미기 탭이 표시된다
- [ ] 꾸미기 탭 안에 이모트 / 배경 / 뱃지 서브탭이 있다
- [ ] 미보유 아이템에 구매 버튼이 표시된다
- [ ] 코인 부족 시 "코인이 부족합니다" 토스트가 표시된다
- [ ] 구매 후 코인이 즉시 차감되고 아이템이 자동 장착된다
- [ ] 보유 아이템에 장착/장착됨 버튼이 올바르게 표시된다
- [ ] 장착 변경 시 로비 프로필과 이모트 설정이 즉시 반영된다
