## 조준 인디케이터 (Green 전용)

- 부채꼴(扇形) 형태, 60° 반경 5 units
- 반투명 초록 채움 + 호(arc) 외곽선 + 양쪽 사이드 라인
- Red의 직사각형 인디케이터와 독립적으로 존재
- 탄약 없거나 장전 중일 때 투명도 낮아짐

## 투사체 동작
1. 클릭 시 4발을 0.08초 간격으로 순차 발사 (index 0 → 1 → 2 → 3)
2. 각 투사체는 `launchAt = gameTime + index * 0.08` 이 될 때까지 대기(비표시)
3. 매 프레임 이동 (`updateProjectiles`) — `launchAt` 미도달 투사체는 스킵
4. 첫 번째 충돌 적에게 거리별 데미지 적용 후 소멸
5. 5 units 도달 시 소멸
6. 벽/수풀 관통 (프로토타입 단계)

## 순차 발사 사양 (신규)

## Regression Guard
- Red 캐릭터의 조준선/공격 판정 변경 없음
- AI Green 봇도 동일한 60° 각도 + 0.08초 순차 발사
- 공격 쿨다운/탄약 소모 로직 변경 없음
