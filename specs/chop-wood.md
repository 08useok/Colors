# 찹 우드 (Chop Wood) — 3v3 팀 벌목전

## 개요
3대3 팀전. 상대 팀의 나무를 먼저 파괴하면 승리.

## 모드 진입
- 로비에 "찹 우드" 버튼 추가 (전투 시작 / 훈련장 옆)
- 캐릭터 선택 후 진입
- 맵: 전용 대칭 맵 (좌/우 팀 영역)

---

## 나무 (Tree)

| 항목 | 값 |
|------|-----|
| 팀당 개수 | 1 |
| 체력 | 100 |
| 공격 피해 | 불가 (투사체/펀치로 데미지 안 됨) |
| 벌목 조건 | 적 나무 반경 3타일 이내 |
| 벌목 주기 | 2초마다 자동 |
| 벌목 데미지 | 도끼 등급에 따라 결정 |

---

### 승급 규칙

**기본 킬 보상**: 플레이어 처치 시 +1 단계

**도끼 흡수 (추가 보상)**:

| 처치한 적 등급 | 추가 승급 |
|---------------|----------|
| 나무~철 | +0 |
| 금 | +2 |
| 다이아 | +2 |
| 에메랄드 | +3 |
| 사파이어 | +3 |
| 루비 | +4 |
| 아메시스트 | +5 |
| 무지개 | +8 |

**사망 패널티**: 도끼 → 나무 도끼로 초기화

---

## 리스폰
- 사망 시 5초 후 자기 팀 스폰 지점에서 부활
- 나무 도끼 상태로 리스폰

---

## 맵 구조 (전용 대칭 맵)

```
[팀A 나무] ---- [중앙 전투 지역] ---- [팀B 나무]
  스폰A                                  스폰B
```

- 맵 크기: 60 x 30 (가로 x 세로)
- 팀A 나무: (-25, 0)
- 팀B 나무: (25, 0)
- 팀A 스폰: (-28, -8~8) 3개 지점
- 팀B 스폰: (28, -8~8) 3개 지점
- 중앙에 벽/수풀 배치 (교전 유도)
- 자기장 없음

## UI

### HUD 추가 요소
- **팀 나무 체력 바**: 화면 상단 좌/우에 각 팀 나무 HP 표시
- **도끼 등급 표시**: 하단 캐릭터 정보 옆에 현재 도끼 아이콘 + 등급명
- **벌목 진행 표시**: 적 나무 근처에 있을 때 "벌목 중..." + 진행 바
- **머리 위 도끼 등급**: 각 플레이어 머리 위에 도끼 색상 아이콘 표시

### 결과 화면
- 승리: "나무를 파괴했습니다!"
- 패배: "나무가 파괴되었습니다..."
- 개인 스탯: 킬, 벌목 데미지, 최고 도끼 등급

---

## 승리 조건
- 상대 나무 체력 0 → 즉시 승리
- 시간 제한 없음 (나무가 파괴될 때까지)

---

## 구현 포인트

### state 확장
```js
state.chopWoodMode = false;
state.teams = { a: { tree: { x, z, health: 100 } }, b: { tree: { x, z, health: 100 } } };
state.playerTeam = "a" | "b";
```

### fighter 확장
```js
fighter.team = "a" | "b";
fighter.axeLevel = 0; // 0=나무, 9=무지개
fighter.isChopping = false;
fighter.chopTimer = 0;
```

### 주요 함수
- `initChopWood()` — 모드 초기화, 팀 배정, 나무 생성
- `updateChopping(dt)` — 벌목 판정 + 데미지
- `onChopWoodKill(attacker, target)` — 도끼 승급 + 흡수
- `respawnFighter(fighter)` — 5초 후 리스폰
- `checkChopWoodEnd()` — 나무 체력 0 체크
- `createChopWoodMap()` — 전용 맵 생성

---

## Regression Guard
- 기존 배틀로얄 모드 영향 없음 (`state.chopWoodMode` 분기)
- 기존 훈련장 영향 없음
- 캐릭터 스탯/공격 시스템 공유 (도끼는 나무에만 적용)

## Acceptance Criteria
- [ ] 로비에서 "찹 우드" 모드 선택 가능
- [ ] 3v3 팀 배정 (플레이어 + 봇 2명 vs 봇 3명)
- [ ] 각 팀 나무 1개, 체력 100
- [ ] 적 나무 3타일 이내 → 2초마다 자동 벌목
- [ ] 도끼 등급별 벌목 피해 적용
- [ ] 킬 시 도끼 승급 + 흡수 시스템 작동
- [ ] 사망 시 도끼 초기화 + 5초 후 리스폰
- [ ] 나무 체력 0 → 승리/패배 화면
- [ ] 머리 위 도끼 등급 표시
- [ ] HUD에 팀 나무 체력 + 도끼 등급 표시
