# Colors 인게임 용어집

게임 기획, 패치 요청, UI 문구에서 같은 대상을 같은 이름으로 부르기 위한 기준 문서다. 오디오 명칭은 [audio-glossary.md](./audio-glossary.md)를 따른다.

## 기본 용어

| 권장 이름 | 뜻 | 혼동하지 말 것 |
|---|---|---|
| 플레이어 | 사람이 직접 조작하는 캐릭터 | 다른 실제 참가자는 `다른 플레이어`라고 표현 |
| AI | 컴퓨터가 조작하는 일반 참가자 | 테이크다운의 `보스`와 구분 |
| 파이터 | 플레이어, 다른 플레이어, AI를 모두 포함하는 전투 개체 | 캐릭터 종류 자체와 구분 |
| 캐릭터 | Red, Green처럼 능력과 기본 능력치가 다른 선택 항목 | 파이터는 실제 경기 안의 개체 |
| 스폰 | 경기 시작 시 파이터가 처음 등장하는 것 | 찹우드 사망 후 등장은 `리스폰` |
| 리스폰 | 사망한 파이터가 일정 시간 후 다시 등장하는 것 | 쇼다운에는 리스폰 없음 |
| 생존자 | 쇼다운에서 아직 사망하지 않은 파이터 | 관전 중인 사망 플레이어 제외 |
| 처치 | 공격으로 상대 체력을 0으로 만든 것 | 자기장 사망과 구분 |
| 사망 | 체력이 0이 되어 전투 불능이 된 상태 | 찹우드·테이크다운에서는 리스폰 가능 |

## 게임 모드

| 권장 이름 | 코드/영문 | 핵심 규칙 | 승리 조건 |
|---|---|---|---|
| 쇼다운 | `showdown` | 실제 플레이어와 AI를 합쳐 10명이 싸우는 개인 생존전 | 마지막 생존자 1위 |
| 찹우드 | `chopwood`, Chop Wood | 3대3 팀전, 처치로 도끼를 강화해 적 나무 벌목 | 적 팀 나무 체력 0 |
| 테이크다운 | `takedown`, Take Down | 제한 시간 동안 보스와 경쟁하는 점수전 | 종료 시 점수가 높은 참가자 |
| 훈련장 | `training` | 선택 캐릭터의 이동·공격을 시험하는 연습 공간 | 승패 없음 |

## 쇼다운 용어

| 권장 이름 | 뜻 |
|---|---|
| 매칭 대기 | 같은 방에 참가할 실제 플레이어를 기다리는 상태 |
| 시작 카운트다운 | 참가자가 들어올 때마다 다시 시작되는 3초 대기 |
| 혼합 매치 | 실제 플레이어와 AI가 함께 참가하는 쇼다운 |
| AI 전용 매치 | 3초 동안 아무도 추가로 참가하지 않아 로컬 플레이어 외 자리를 AI로 채운 경기 |
| 스폰 슬롯 | 맵 가장자리에 배치된 10개의 시작 위치 |
| 자기장 | 시간이 지나면서 안전 지역을 줄이고 바깥 파이터에게 피해를 주는 영역 |
| 안전 지역 | 자기장 피해를 받지 않는 원 내부 |
| 자기장 단계 | 대기 후 반경과 피해량이 달라지는 축소 구간 |
| 최후의 쇼다운 | 생존자가 정확히 2명 남은 상태 |
| 순위 | 사망 순서로 정해지는 1위~10위 결과 |
| 관전 | 사망 후 남은 AI의 전투를 카메라로 보는 상태 |
| 관전 종료 | 남은 전투를 기다리지 않고 현재 순위로 결과창을 여는 기능 |
| 우승 줌 | 1위 캐릭터를 가까이 보여주며 약 2초간 전투를 정지하는 연출 |
| AI 우승 연출 | 플레이어가 2위이고 AI가 1위일 때 AI에게 적용되는 우승 줌 |

## 찹우드 용어

| 권장 이름 | 뜻 |
|---|---|
| 아군 나무 | 플레이어 팀이 보호해야 하는 나무 |
| 적 나무 | 먼저 파괴해야 하는 상대 팀 나무 |
| 벌목 | 적 나무 근처에서 진행 게이지를 채워 나무에 피해를 주는 행동 |
| 벌목 피해 | 도끼 등급에 따라 적 나무에 주는 피해 |
| 도끼 등급 | 나무부터 무지개까지 이어지는 10단계 벌목 능력 |
| 도끼 승급 | 적 처치 후 도끼 등급이 상승하는 것 |
| 도끼 흡수 | 처치한 적의 높은 도끼 등급에 따라 추가 승급하는 보상 |
| 사망 패널티 | 사망 시 도끼가 나무 등급으로 초기화되는 규칙 |
| 팀 리스폰 | 사망 5초 후 아군 스폰 지역에서 부활하는 것 |
| 찹우드 전적 | 쇼다운과 별도로 저장되는 찹우드 승수·판수·승률 |
| 시즌 찹우드 전적 | 시즌별 전체 및 캐릭터별 찹우드 기록 |

## 테이크다운 용어

| 권장 이름 | 뜻 |
|---|---|
| 보스 | 중앙 전장에서 싸우는 고체력 AI 대상 |
| 보스 피해 점수 | 보스에게 준 피해를 기반으로 얻는 점수 |
| 막타 보너스 | 보스의 체력을 마지막으로 0으로 만든 참가자가 얻는 추가 점수 |
| 처치 점수 | 다른 참가자를 처치해 얻는 점수 |
| 제한 시간 | 테이크다운 경기가 끝날 때까지 남은 시간 |
| 테이크다운 리스폰 | 사망 5초 후 다시 참가하는 것 |
| 방향 표시기 | 화면 밖 보스의 방향을 알려주는 HUD |
| 전술 지도 | 테이크다운 전장의 전체 배치를 보여주는 지도 UI |
| 자기장 없음 | 테이크다운에서는 자기장 계산·피해·표시를 사용하지 않는 규칙 |

## 전투 용어

| 권장 이름 | 뜻 |
|---|---|
| 기본 공격 | 캐릭터가 탄약을 소비해 사용하는 주 공격 |
| 탄약 | 기본 공격에 필요한 횟수 자원 |
| 자동 재장전 | 공격하지 않는 동안 탄약이 자동으로 회복되는 시스템 |
| 공격 쿨다운 | 다음 공격을 시작하기 전까지의 최소 간격 |
| 사거리 | 공격이 도달하거나 판정되는 최대 거리 |
| 투사체 | 총알, 부메랑, 폭탄처럼 공간을 이동하는 공격 개체 |
| 직격 피해 | 투사체가 대상에 직접 맞을 때 주는 피해 |
| 범위 피해 | 폭발이나 원형 판정 안의 여러 대상에게 주는 피해 |
| 지속 피해 | 독처럼 일정 시간에 걸쳐 반복 적용되는 피해 |
| 감속 | 일정 시간 이동 속도를 낮추는 상태 효과 |
| 회복 | 잃은 체력을 되돌리는 효과 |
| 자연 회복 | 최근 전투 후 일정 시간이 지나면 자동으로 체력을 회복하는 시스템 |
| 아군 공격 방지 | 팀 모드에서 아군에게 피해를 주지 않는 규칙 |
| 발각 | 수풀 안에 있어도 공격·피격 후 일정 시간 상대에게 보이는 상태 |
| 조준 표시기 | 현재 공격 방향과 범위를 바닥에 보여주는 UI |

## 캐릭터 역할과 공격 명칭

| 캐릭터 | 권장 역할 | 기본 공격 명칭 | 핵심 특징 |
|---|---|---|---|
| Red | 근접 탱커 | 더블 펀치 | 높은 체력, 근거리 연속 타격 |
| Green | 중거리 기동형 | 부메랑 | 여러 발의 부메랑과 거리별 피해 |
| Blue | 장거리 저격수 | 저격탄 | 긴 사거리, 낮은 체력 |
| Orange | 범위 공격수 | 폭탄 | 직격 후 5갈래 폭발 |
| Yellow | 제어형 공격수 | 전기 투사체 | 감전과 이동 속도 감소 |
| Cyan | 중거리 확산형 | 확산 직선탄 | 나란히 퍼지는 다중 투사체 |
| Purple | 지속 피해형 | 독침·약병 | 독침과 범위 독 피해 |
| Pink | 근접 지원가 | 치유 원 | 적 피해와 아군 회복을 함께 수행 |

## 계정·결과 용어

| 권장 이름 | 뜻 |
|---|---|
| 트로피 | 경기 순위와 연승 보너스로 증감하는 경쟁 점수 |
| 트로피 변화량 | 한 경기 결과로 얻거나 잃는 트로피 수 |
| 승리 | 기록상 승리로 처리되는 결과. 쇼다운은 4위 이내, 찹우드는 팀 승리 |
| 1위 | 쇼다운의 마지막 생존자. 일반 승리와 구분되는 최상위 결과 |
| 연승 | 승리 판정이 연속으로 이어진 횟수 |
| 최고 연승 | 계정에서 달성한 가장 높은 연승 기록 |
| 승수 | 승리 판정을 받은 경기 수 |
| 판수 | 해당 모드 또는 캐릭터로 완료한 전체 경기 수 |
| 승률 | `승수 ÷ 판수 × 100` |
| 전체 전적 | 모든 모드가 반영된 계정 승리·패배 기록 |
| 캐릭터 전적 | 특정 캐릭터로 플레이한 승수·판수·승률 |
| 시즌 전적 | 현재 시즌과 이전 시즌을 분리한 기록 |
| 코인 | 경기 보상과 상점 구매에 사용하는 화폐 |
| 크레딧 | 캐릭터 구매에 사용하는 별도 화폐 |

## 로비·UI 용어

| 권장 이름 | 뜻 |
|---|---|
| 로비 | 계정, 캐릭터, 모드, 상점 등을 선택하는 메인 화면 |
| 내 정보 | 트로피와 전체·캐릭터·시즌 전적을 보는 패널 |
| 리더보드 | 다른 플레이어를 포함한 트로피 순위표 |
| 캐릭터 선택 | 플레이할 캐릭터를 고르는 화면 |
| 착용 | 보유한 스킨·이모트·배경·배지를 실제 사용 상태로 지정하는 것 |
| 보유 | 아이템을 계정이 소유하고 있지만 반드시 착용한 것은 아닌 상태 |
| 이모트 슬롯 | 숫자키 `1`, `2`, `3`에 대응하는 세 개의 착용 위치 |
| HUD | 경기 중 체력, 탄약, 생존자, 자기장 등을 표시하는 화면 정보 |
| 킬 피드 | 누가 누구를 처치했는지 잠시 표시하는 목록 |
| 결과창 | 순위, 트로피, 코인, 피해량 등을 표시하는 경기 종료 화면 |

## 요청할 때 권장하는 문장

- "쇼다운의 스폰 슬롯을 더 멀리 배치해줘."
- "최후의 쇼다운이 시작되면 자기장 피해를 높여줘."
- "Blue의 저격탄 사거리를 10% 줄여줘."
- "찹우드 벌목 피해에는 캐릭터 공격력이 적용되지 않게 해줘."
- "테이크다운 보스 피해 점수를 절반으로 줄여줘."
- "캐릭터별 시즌 찹우드 승률을 내 정보에 표시해줘."

모호한 `능력`, `점수`, `음악`, `시작 위치` 대신 이 문서의 `기본 공격`, `트로피/처치 점수`, 오디오 용어, `스폰 슬롯`처럼 범위가 명확한 이름을 사용한다.
