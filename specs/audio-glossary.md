# Colors 오디오 용어집

게임 기획, 개발 요청, 코드에서 오디오 이름을 일관되게 사용하기 위한 기준 문서다.

## 가장 중요한 구분

| 구분 | 뜻 | 재생 방식 |
|---|---|---|
| BGM | 일정 시간 계속 흐르는 배경 음악 | MP3 파일 반복 재생 |
| SFX | 사건이 발생할 때 짧게 한 번 나오는 효과음 | Web Audio 실시간 합성 |
| 스팅어 | 장면 전환이나 중요한 순간을 알리는 매우 짧은 음악형 효과음 | 현재는 SFX로 합성 |
| 루프 | 음악이 끝나면 처음부터 다시 재생되는 설정 | `Audio.loop = true` |
| 오디오 활성화 | 브라우저 정책상 사용자가 화면을 한 번 누른 뒤 소리를 재생할 수 있는 상태 | `state.audioEnabled` |

## 공식 BGM 명칭

앞으로 요청할 때는 아래의 **권장 이름**을 사용한다.

| 권장 이름 | 코드 이름 | 파일 | 음량 | 시작 조건 | 종료 조건 |
|---|---|---|---:|---|---|
| 로비 BGM | `lobbyBgm` | `assets/lobby-bgm.mp3` | 30% | 오디오가 활성화된 상태에서 로비 진입 | 전투·훈련 진입 |
| 쇼다운 전투 BGM | `showdownBgm` | `assets/showdown-bgm.mp3` | 35% | 쇼다운 경기 시작 | 생존자 2명 또는 경기 종료 |
| 최후의 쇼다운 BGM | `showdownMusic` | `assets/showdown-theme.mp3` | 60% | 생존자가 정확히 2명이 되는 순간 | 경기 종료 또는 로비 이동 |

### 혼동하기 쉬운 표현

- `쇼다운 전투 BGM`은 쇼다운 모드의 평상시 전투 음악이다.
- `최후의 쇼다운 BGM`은 쇼다운 모드에서 마지막 2명이 남았을 때 바뀌는 음악이다.
- 코드의 `showdownMusic`이라는 이름은 모호하므로 기획 대화에서는 **최후의 쇼다운 BGM**이라고 부른다.
- 현재 별도의 `승리 BGM` MP3는 없다. 승리 때 들리는 소리는 짧은 **승리 SFX**다.

## 주요 SFX 명칭

| 권장 이름 | 코드 호출 | 발생 조건 | 길이/성격 |
|---|---|---|---|
| 게임 시작 SFX | `audio.play("gameStart")` | 쇼다운·찹우드·테이크다운·훈련 시작 | 약 0.3초 상승음 |
| 최후의 쇼다운 스팅어 | `audio.play("showdownStart")` | 생존자가 정확히 2명이 되는 순간 | 약 0.7초 긴장음 |
| 승리 SFX | `audio.play("win")` | 쇼다운 1위 또는 찹우드 승리 | 약 0.8초 상승 팡파르 |
| 패배 SFX | `audio.play("lose")` | 플레이어 사망 또는 모드 패배 | 약 0.7초 하강음 |
| 매칭 발견 SFX | `audio.play("matchFound")` | 방 참가자 입장 | 짧은 확인음 |
| 카운트다운 SFX | `audio.play("countdownTick")` | 시작 카운트 숫자가 바뀔 때 | 짧은 비프음 |
| 자기장 축소 SFX | `audio.play("zoneShrink")` | 자기장 단계 전환 | 하강 경고음 |
| 자기장 피해 경고 SFX | `audio.play("warning")` | 플레이어가 자기장 밖에 있을 때 | 반복 경고음 |
| 처치 SFX | `audio.play("kill")` | 플레이어가 관여한 처치 | 짧은 처치 확인음 |
| 피격 SFX | `audio.play("damaged")` | 로컬 플레이어가 피해를 받음 | 둔탁한 하강음 |

## 최후의 쇼다운 전환 순서

생존 인원이 `3명 → 2명`으로 바뀌는 프레임에 다음 순서로 실행된다.

1. `쇼다운 전투 BGM` 정지
2. `최후의 쇼다운 스팅어` 1회 재생
3. `최후의 쇼다운 BGM`을 처음부터 반복 재생
4. 화면에 쇼다운 안내 문구를 약 1.8초 표시

## 결과 연출 용어

| 용어 | 뜻 |
|---|---|
| 1위 확정 | 마지막 생존자가 결정된 순간 |
| 우승 줌 | 우승 캐릭터를 약 2초간 가까이 보여주는 카메라 연출 |
| 결과 대기 | 우승 줌 또는 사망 연출이 끝나기 전 결과창을 숨기는 시간 |
| 결과창 | 순위, 트로피 변화, 코인, 피해량을 표시하는 화면 |
| AI 우승 연출 | 플레이어가 2위이고 AI가 1위일 때 AI에게 적용하는 우승 줌 |

## 요청 문장 예시

- "로비 BGM 음량을 20%로 줄여줘."
- "쇼다운 전투 BGM을 다른 파일로 교체해줘."
- "생존자 2명부터 최후의 쇼다운 BGM을 재생해줘."
- "최후의 쇼다운 스팅어를 0.5초로 줄여줘."
- "승리 SFX가 우승 줌 시작과 동시에 나오게 해줘."

`쇼다운 음악`, `시작 음악`, `승리 음악`처럼 여러 의미로 해석될 수 있는 표현 대신 위 권장 이름을 사용한다.
