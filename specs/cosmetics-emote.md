# 인게임 이모트

## 개요

전투 중 Q키 또는 화면 버튼으로 이모트를 발동하면, 자기 캐릭터 위에 이모지 풍선이 2.5초간 표시된다.

---

## 아이템 정의 (5개)

| id | 이모지 | 이름 | 가격(코인) |
|---|---|---|---|
| `emote_thumbs` | 👍 | 엄지척 | 300 |
| `emote_laugh` | 😂 | 웃음 | 300 |
| `emote_cool` | 😎 | 선글라스 | 300 |
| `emote_fire` | 🔥 | 불꽃 | 300 |
| `emote_skull` | 💀 | 해골 | 300 |

---

## 발동 조건

- 전투 화면(`state.gameStarted === true`)에서만 사용 가능
- Q키 또는 화면 우하단 이모트 버튼
- 장착된 이모트가 없으면 버튼/키 비활성

---

## 쿨다운

- 발동 후 5초 쿨다운
- 쿨다운 중 버튼에 남은 시간 표시 (초 단위)
- 사망 후에도 쿨다운 유지

---

## 이모지 풍선 렌더링

- Three.js `Sprite` + `CanvasTexture`로 이모지 텍스트를 canvas에 그려 스프라이트에 적용
- 위치: 캐릭터 머리 위 `y = 3.0`
- 크기: 1.2 × 1.2 world unit
- 2.5초 후 자동 소멸 (fade-out 0.3초)
- 동시에 여러 이모트가 겹치지 않도록 자기 캐릭터 1개만 표시

---

## 계정 스키마 추가

```json
"cosmetics": {
  "ownedEmotes": ["emote_thumbs"],
  "equippedEmote": "emote_thumbs"
}
```

- `ownedEmotes`: 보유 중인 이모트 id 배열
- `equippedEmote`: 현재 장착한 이모트 id (null = 없음)

---

## 장착 UI

- 상점 → 이모트 탭에서 보유 아이템에 "장착" 버튼 표시
- 장착 시 기존 장착 이모트 해제 (1개만 장착 가능)

---

## Acceptance Criteria

- [ ] Q키 입력 시 내 캐릭터 위에 이모지 풍선이 표시된다
- [ ] 이모지 풍선은 2.5초 후 사라진다
- [ ] 발동 후 5초 동안 버튼/키가 비활성화된다
- [ ] 장착된 이모트가 없으면 Q키/버튼이 동작하지 않는다
- [ ] 전투 화면 밖에서는 Q키가 이모트를 발동하지 않는다
- [ ] 장착된 이모트 변경 시 즉시 반영된다
