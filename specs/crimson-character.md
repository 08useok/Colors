# 크림슨 (Crimson) — 신규 캐릭터

## 1. 기본 정보

| 항목 | 값 |
|---|---|
| 이름 | 크림슨 |
| 역할 | 근접 브루저 (난전 / 한타 브레이커) |
| 등급 | 영웅 (`specs/character-rarity-shop.md` 참조, 900크레딧) |
| 출시 시즌 | 베타 시즌 1 |
| 이동 속도 배율 | 매우 빠름 (기존 캐릭터 중 최고 — Pink 1.4보다 높게 설정) |
| 체력 | 9,800 |
| 사거리 | 4타일 (근접) |
| 색상 | 진한 빨강 계열 (기존 Red `#ff0000`와 구분되는 톤, 예: `#a00000`) |

## 2. 일반 공격 — 부채꼴 4연타

- 공격 방식: 전방 부채꼴 범위에 4연속 근접 펀치
- 부채꼴 범위 내 여러 적을 동시에 타격 가능
- 근접할수록 명중률/효율이 높아짐 (기존 Red 더블 펀치와 동일한 근접 판정 방식)

| 타수 | 피해량 |
|---|---|
| 1타 | 600 |
| 2타 | 700 |
| 3타 | 800 |
| 4타 | 1100 |
| 4타 모두 명중 시 총합 | 3200 |

### 타이밍/판정 (제안값 — Green 부메랑 순차 발사 패턴 참고)
| 항목 | 값 |
|---|---|
| 부채꼴 각도 | 90° (-45°, -15°, +15°, +45° 방향에 순차 판정) |
| 타격 간격 | 0.12초 (1타→2타→3타→4타 순차) |
| 공격 사거리 | 4타일 (근접) |
| attackCooldown (콤보 종료 후 재사용 대기) | 0.6s |
| reloadDuration | 0.5s (기존 캐릭터 공통 패턴) |

- 일반 공격 명중 시마다 궁극기 게이지 1획 충전 — 8회 명중 시 궁극기 사용 가능 (풀 충전)

## 3. 궁극기 — 파괴의 일격

| 항목 | 값 |
|---|---|
| 피해량 | 3000 |
| 범위 | 5×5 타일 |
| 벽 파괴 | 범위 내 벽을 영구 제거 (이동/충돌 판정에서도 완전히 사라짐, 전투 종료까지 복구되지 않음) |
| 넉백 | 범위 내 적을 1~2타일 밀쳐냄 |
| 조준 | 캐릭터가 바라보는 정면 방향 고정 (마우스 조준과 무관하게 현재 이동/정지 시 바라보는 방향 기준으로 앞쪽에 5×5 범위 생성) |
| 발동 조건 | 궁극기 게이지 풀 충전(일반 공격 8회 명중) |

### 궁극기 게이지 규칙
- 게이지는 일반 공격이 적에게 명중할 때마다 1씩 증가 (최대 8)
- **사망해도 게이지는 초기화되지 않고 유지된다** (리스폰 후에도 기존 충전량 그대로)
- 게이지가 8에 도달하면 즉시 사용 가능 상태가 되고, 발동 후 0으로 리셋
- HUD에 게이지 충전량을 표시 (예: 기존 재장전 바와 유사한 형태의 진행 바, "8/8"에서 발동 가능 표시)

### 벽 파괴 적용 범위
- 궁극기의 5×5 범위 안에 있는 벽(`state.solids`/`state.battleSolids`의 `type: "wall"` 오브젝트)을 배열에서 제거하고 3D 오브젝트도 씬에서 제거
- 제거된 벽은 봇 경로탐색(`isWallAhead`, `findWallEscapeDir`)에서도 즉시 장애물로 인식되지 않아야 함
- 훈련장(`trainingSolids`)과 찹 우드 모드(`battleSolids`)에서도 동일하게 동작

## 4. 봇 AI
- AI 봇도 크림슨으로 배정될 수 있다 (`ROTATION_CHAR_ORDER`, `botTypes` 등 기존 캐릭터 배열에 `"crimson"` 추가)
- 봇은 근접 사거리(4타일) 특성상 Red와 유사하게 적극적으로 근접하는 성향으로 동작
- 봇도 궁극기 게이지가 차면 사용한다 (사용 시점 로직은 근접 교전 중 우선 사용)

## 5. 계정/통계 연동
- `charStats.crimson`, `seasonCharStats[시즌].crimson` 초기화 항목 추가 (기존 8캐릭터와 동일한 `{ wins, games }` 구조)
- 캐릭터 선택 화면에 크림슨 버튼 추가 — 단, 등급 시스템에 따라 미보유 시 잠금 표시 (`specs/character-rarity-shop.md` 참조)
- 상성표(`specs/character-stats.md`)에는 이번 시즌 범위 밖 — 크림슨 상성 데이터는 비목표로 둔다

## Non-goals
- 궁극기 게이지를 다른 기존 캐릭터에 적용하는 것 (이번 시즌은 크림슨 전용)
- 벽 파괴에 따른 투사체 통과 로직 변경 (기존에도 투사체는 벽과 충돌하지 않음 — 이 비목표는 유지)
- 궁극기 전용 처치 시 스코어/트로피 보너스

## Acceptance Criteria
- [ ] 로비 캐릭터 선택에 크림슨이 표시되고, 보유 시 선택/전투 진입이 가능하다
- [ ] 크림슨 기본 공격은 부채꼴 4연타로 동작하며 각 타수 피해량이 600/700/800/1100이다
- [ ] 부채꼴 범위 내 여러 적이 동시에 피해를 입는다
- [ ] 일반 공격 명중 8회 시 궁극기 게이지가 가득 차고 사용 가능 상태가 된다
- [ ] 사망 후 리스폰해도 궁극기 게이지 충전량이 유지된다
- [ ] 궁극기 발동 시 캐릭터 정면 방향으로 5×5 범위에 3000 피해가 적용된다
- [ ] 궁극기 범위 내 벽이 영구히 제거되고, 제거 즉시 이동/봇 경로탐색에서 장애물로 인식되지 않는다
- [ ] 궁극기 범위 내 적이 1~2타일 밀쳐진다
- [ ] 궁극기 사용 후 게이지가 0으로 초기화된다
- [ ] AI 봇이 크림슨으로 스폰·전투·궁극기 사용을 정상 수행한다
- [ ] HUD에 크림슨의 궁극기 게이지가 표시된다
