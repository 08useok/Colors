# 로비 화면

## 흐름

```
최초:       로드 → 회원가입(아이디+닉네임) → 로비 메인
재방문(첫): 로드 → 일일 아이디 인증 → 로비 메인
재방문(후): 로드 → 로비 메인 (인증 생략)
게임 중:    로비 메인 → 캐릭터 선택 → 전투 시작 → 게임
게임 후:    게임 종료 → 전적/트로피 갱신 → 로비 복귀
```

→ 상세 인증 로직: `specs/daily-login.md` 참조

## 로비 UI 구성

### 계정 생성 화면 (최초 1회)
- 아이디 입력 (최대 16자) + 닉네임 입력 (최대 12자)
- `전장 입장` 버튼 → 둘 다 1자 이상이어야 동작, 저장 후 로비 메인으로 전환

### 일일 인증 화면 (매일 첫 접속)
- 아이디 입력 후 `확인` → 맞으면 로비 메인
- 틀리면 "아이디가 틀렸습니다." + 남은 시도 횟수
- 5회 초과 시 "계정 문제 해결하기" 버튼 → 아이디 표시

### 로비 메인 화면
1. **계정 헤더** — 닉네임, 레벨, 트로피 수, 전적(N승 N패)
2. **캐릭터 선택** — Red / Green 버튼 (기존 스타일 유지, 선택 시 테두리 강조)
3. **전투 시작 버튼** — 캐릭터 선택 후 활성화

## 트로피 / 레벨 규칙

순위 기반 분배 (→ `specs/trophy-ranking.md` 참조):

| 순위 | 트로피 |
|---|---|
| 1위 | +10 |
| 2위 | +8 |
| 3위 ~ 5위 | +6 ~ +2 |
| 6위 | 0 |
| 7위 ~ 10위 | -2 ~ -8 |

- 최솟값 0
- 레벨 = `Math.floor(trophies / 300) + 1`
- 레벨은 표시 전용, 게임 플레이에 영향 없음

## Acceptance Criteria
- [ ] 최초 로드 시 닉네임 입력 화면이 표시된다
- [ ] 재방문 시 로비 메인이 바로 표시된다
- [ ] 캐릭터 선택이 로비에서 이루어진다
- [ ] 게임 종료 후 트로피/전적이 갱신된다
- [ ] 다시 시작 시 로비 메인으로 복귀한다
- [ ] 선택된 캐릭터가 시각적으로 강조된다
