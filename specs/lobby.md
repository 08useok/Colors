# 로비 화면

## 흐름

```
로드 → (계정 없으면) 닉네임 입력 → 로비 메인
로비 메인 → 캐릭터 선택 → 전투 시작 → 게임
게임 종료 → 전적/트로피 갱신 → 로비 복귀
```

## 데이터 스키마 (localStorage: `skullCreekAccount`)

```json
{
  "nickname": "플레이어",
  "trophies": 0,
  "wins": 0,
  "losses": 0,
  "selectedCharacter": "red"
}
```

## 로비 UI 구성

### 계정 생성 화면 (최초 1회)
- 닉네임 입력 (최대 12자, 영문/한글/숫자)
- `시작하기` 버튼 → 저장 후 로비 메인으로 전환

### 로비 메인 화면
1. **계정 헤더** — 닉네임, 레벨, 트로피 수, 전적(N승 N패)
2. **캐릭터 선택** — Red / Green 버튼 (기존 스타일 유지, 선택 시 테두리 강조)
3. **전투 시작 버튼** — 캐릭터 선택 후 활성화

## 트로피 / 레벨 규칙

| 결과 | 트로피 |
|---|---|
| 승리 | +30 |
| 패배 | -10 (최솟값 0) |

- 레벨 = `Math.floor(trophies / 300) + 1`
- 레벨은 표시 전용, 게임 플레이에 영향 없음

## Acceptance Criteria
- [ ] 최초 로드 시 닉네임 입력 화면이 표시된다
- [ ] 재방문 시 로비 메인이 바로 표시된다
- [ ] 캐릭터 선택이 로비에서 이루어진다
- [ ] 게임 종료 후 트로피/전적이 갱신된다
- [ ] 다시 시작 시 로비 메인으로 복귀한다
- [ ] 선택된 캐릭터가 시각적으로 강조된다
