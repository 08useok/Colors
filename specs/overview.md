# 해골천 (Skull Creek) — 프로젝트 개요

## 목표
브라우저에서 즉시 실행 가능한 3D TPS 배틀로얄 프로토타입.
브롤스타즈 직관성을 탑다운 3D 감각으로 구현한다.

## 기술 스택
- **렌더링**: Three.js r165 (CDN, 번들러 없음)
- **언어**: Vanilla JS ES Module, 단일 `src/main.js`
- **스타일**: `styles.css`
- **오디오**: Web Audio API (합성음, 에셋 없음)
- **멀티플레이**: 미구현 (로컬 AI 봇 9명)

## 주요 기능
1. 탑다운 시점 TPS 전투
2. 마우스 조준 + WASD 이동 분리
3. 캐릭터 선택 (Red / Green)
4. 자기장 5단계 축소
5. AI 봇 9명 (Red/Green 랜덤 배정)

## Non-goals
- 실시간 멀티플레이 (서버 없음)
- 무기 교체 / 파밍 / 인벤토리
- 점프 / 대시 / 슬라이딩 / 헤드샷
- 투사체의 벽 충돌 (프로토타입 단계)
