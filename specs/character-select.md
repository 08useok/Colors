# 캐릭터 선택 시스템

## UI 흐름
1. 게임 로드 → 캐릭터 선택 화면 표시 (기존 인트로 화면에 통합)
2. Red 또는 Green 버튼 클릭 → `state.selectedCharacter` 저장 → 게임 시작
3. 재시작 시 → 캐릭터 선택 화면으로 복귀

## 캐릭터 정의

| 항목 | Red | Green |
|---|---|---|
| HP | 10,000 | 8,400 |
| 공격 | 더블 펀치 | 부메랑 4연발 |
| 사거리 | 4타일 | 5타일 |
| 색상 hex | `#e53729` | `#3dbd4a` |

## AI 봇
- 9명의 AI 봇이 게임 시작 시 Red / Green 50:50 확률로 배정
- 봇의 공격 방식은 캐릭터 타입에 따라 자동 결정

## Acceptance Criteria
- [ ] 게임 시작 전 캐릭터 선택 화면이 표시된다
- [ ] Red 선택 시 HP 10,000, 더블 펀치로 플레이
- [ ] Green 선택 시 HP 8,400, 부메랑 4연발로 플레이
- [ ] AI 봇이 Red/Green 혼합으로 등장한다
- [ ] 재시작 시 캐릭터 선택 화면이 다시 표시된다
