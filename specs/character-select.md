# 캐릭터 선택 시스템

## UI 흐름
1. 게임 로드 → 로비 화면 (계정 생성 또는 일일 인증 후)
2. 캐릭터 버튼 클릭 → `state.selectedCharacter` 저장
3. "전투 시작" 또는 "찹 우드" 클릭 → 게임 시작
4. 재시작 시 → 로비로 복귀

## AI 봇
- 9명의 AI 봇이 게임 시작 시 9캐릭터(Crimson 포함) 중 랜덤 배정
- 봇의 공격 방식은 캐릭터 타입에 따라 자동 결정
- 봇 배정은 플레이어의 캐릭터 보유 여부와 무관 (`specs/character-rarity-shop.md` 참조)
