# Skull Creek Prototype

브라우저에서 실행되는 3D TPS 배틀로얄 프로토타입입니다.

## 실행

PowerShell에서 아래 명령으로 간단한 로컬 서버를 띄운 뒤 브라우저에서 접속합니다.

```powershell
cd "C:\Users\useok\OneDrive\문서\New project"
py -m http.server 4173
```

그 다음 브라우저에서 아래 주소를 엽니다.

```text
http://localhost:4173
```

## 조작

- `WASD`: 방향 이동
- `클릭/우클릭`: 더블 펀치
- 탄약 소진 시 자동 장전
- 웹: 좌측 조이스틱을 마우스로 드래그해 보조 이동 가능
- 모바일: 좌측 하단 조이스틱으로 이동
- 모바일: 우측 하단 `공격` 버튼으로 공격

시점은 완전 탑다운 상단 시점으로 설정되어 있습니다.

## 포함된 시스템

- 3인칭 탑다운 전투 카메라
- 더블 펀치 2연타 공격
- 탄약 3발과 자동 장전
- 반동과 안정성 저하
- 체력과 처치 처리
- 자기장 축소 및 장외 피해
- 간단한 AI 봇 9명
- 해골천 스타일 사막형 맵
- HUD, 킬 피드, 결과 화면
